Tips & Tricks
✨
1. Direct Copy First select the text natively on any webpage to copy it instantly.
⬚
2. Selected Area Use the Selected Area tool and draw a box to copy any page text block.
🔗
3. Copy Link Text Use the Copy from Link tool to click any hyperlink and copy its text content.
🖼️
4. Copy from Image Use the Copy from Image tool to draw a box over any image area to extract text.
🎥
5. Copy from Video Pause the video, click the Copy from Image tool, then draw a box around the video text area.
🔑
6. New Google API Key If the API limit is reached, generate and enter a new Gemini API Key from Google AI Studio.
